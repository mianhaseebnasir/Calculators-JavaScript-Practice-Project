<h1>Calculators – JavaScript Practice Project</h1>
<hr>
<h3>📌 Project Overview :</h3>
Calculators is a simple JavaScript-based project designed to demonstrate the fundamentals of connecting HTML with JavaScript to perform calculation-related operations.

This project serves as a beginner-friendly example for understanding how web pages interact with JavaScript files and how logic can be implemented to build calculator functionality.

The project structure is intentionally kept clean and simple so that learners can easily understand the workflow between HTML structure and JavaScript logic.
<hr>
<h3>🚀 Features :</h3>
<li>Simple and clean HTML boilerplate structure</li>
<li>External JavaScript file integration</li>
<li>Beginner-friendly code structure</li>
<li>Lightweight and easy to understand</li>
<li>Ideal for JavaScript practice and learning</li>
<hr>
<h3>📂 Project Structure :</h3>
Calculators/<br>
│<br>
├── index.html      # Main HTML file<br>
├── script.js       # JavaScript logic for calculator<br>
└── README.md       # Project documentation<br>
<hr>
<h3>🛠️ Technologies Used :</h3>
<li>HTML5 – Page structure</li>
<li>JavaScript (Vanilla JS) – Calculator logic and functionality</li>
<hr>
<h3>📖 How It Works :</h3>
<li>The HTML file provides the main structure of the web page.</li>
<li>A JavaScript file (script.js) is linked to the HTML file.</li>
<li>JavaScript handles the logic required for calculator operations.</li>
<li>When the page runs in a browser, the JavaScript executes and enables the calculation functionality.</li>
<hr>
<h3>💻 How to Run the Project :</h3>
<li>Download or clone the repository</li>
<li>Open the project folder</li>
<li>Run the index.html file in any web browser</li>
